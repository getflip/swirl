Get started by setting up your dev environment. `Node.js v16+`, `Git` and
`Yarn v1` must be installed on your machine.

1. **Clone the repository**

```bash
git clone git@github.com:flip-corp/swirl.git
```

The Swirl Web Components Library is located at `packages/swirl-components`.

2. **Install dependencies**

```bash
yarn
```

3. **Get Stencil.js and Storybook up and running**

```bash
cd packages/swirl-components
yarn start
```

Storybook will open in your default browser. Visit
[http://localhost:6006](http://localhost:6006) if it doesn't.
