## Theming

| Variable                        | Usage                                     |
| ------------------------------- | ----------------------------------------- |
| `--s-surface-overlay-default`   | Background color of the picker            |
| `--s-text-default`              | Text color of the picker                  |
| `--s-text-subdued`              | Text color of weekdays and inactive dates |
| `--s-surface-overlay-hovered`   | Background color of the hovered date      |
| `--s-surface-overlay-pressed`   | Background color of the active date       |
| `--s-surface-highlight-default` | Background color of the selected date     |
| `--s-text-on-surface-highlight` | Text color of the selected date           |
| `--s-border-highlight`          | Border color of current date              |
| `--s-icon-highlight`            | Icon color of the month steppers          |
