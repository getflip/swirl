## Accessibility

### ARIA

The component follows the
[WAI-ARIA Table Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/table/).

| ARIA                  | Usage                   |
| --------------------- | ----------------------- |
| `role="columnheader"` |                         |
| `aria-sort`           | Set on a sorted column. |

### Keyboard

No component-specific keyboard behavior.
