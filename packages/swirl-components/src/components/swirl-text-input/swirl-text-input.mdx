# SwirlTextInput

The SwirlTextInput component is used to render a text input field for user input
of various types. It should always be used in combination with the
[SwirlFormControl component](?path=/docs/components-swirlformcontrol--docs).

## Accessibility

### ARIA

The input and controls expose ARIA for form and autocomplete semantics.

| ARIA                | Usage                                                                 |
| ------------------- | --------------------------------------------------------------------- |
| `aria-autocomplete` | Optional; set via `swirlAriaAutocomplete`.                            |
| `aria-controls`     | Optional; set via `swirlAriaControls`.                                |
| `aria-describedby`  | Optional; set via `swirlAriaDescribedby`.                            |
| `aria-disabled`     | Set when the input is disabled.                                      |
| `aria-expanded`     | Optional; set via `swirlAriaExpanded` (e.g. for dropdowns).           |
| `aria-invalid`      | Set when the value is invalid.                                       |
| `role`              | Optional; set via `swirlRole`.                                       |
| `aria-label`        | Set on the clear and password toggle buttons.                        |
| `aria-live`         | Set to `"polite"` on the character counter when present.              |
| `aria-hidden`       | Set on decorative prefix/suffix content.                             |

### Keyboard

Standard text input behavior. Clear and password toggle buttons are activated with <kbd>Enter</kbd> or <kbd>Space</kbd>. No component-specific shortcuts.
