# SwirlTextInput

The SwirlTextInput component is used to render a text input field for user input
of various types. It should always be used in combination with the
[SwirlFormControl component](?path=/docs/components-swirlformcontrol--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=545%3A13417)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-text-input)**

## Theming

| Variable               | Usage                                  |
| ---------------------- | -------------------------------------- |
| `--s-text-default`     | Text color of the input value          |
| `--s-text-disabled`    | Text color of the disabled input value |
| `--s-text-subdued`     | Text color of the suffix and prefix    |
| `--s-icon-default`     | Icon color of the clear button         |
| `--s-border-highlight` | Caret color                            |
