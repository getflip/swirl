## Theming

| Variable                     | Usage                                           |
| ---------------------------- | ----------------------------------------------- |
| `--s-text-default`           | Text color of the item                          |
| `--s-text-highlight`         | Text color of the active item                   |
| `--s-icon-default`           | Icon color of the active item                   |
| `--s-icon-highlight`         | Icon color of the item                          |
| `--s-background-default`     | Background color of the item                    |
| `--s-background-hovered`     | Background color of the hovered item            |
| `--s-background-pressed`     | Background color of the pressed item            |
| `--s-surface-raised-default` | Background color of the active item             |
| `--s-surface-raised-hovered` | Background color of the active and hovered item |
| `--s-surface-raised-pressed` | Background color of the active and pressed item |
