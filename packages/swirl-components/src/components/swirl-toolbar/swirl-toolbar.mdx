# SwirlToolbar

The SwirlToolbar component is used to provide a set of actions that are relevant
for the given context. It has to be used in conjunction with the
[SwirlButton](?path=/docs/components-swirlbutton--docs) and / or
[SwirlChip](?path=/docs/components-swirlchip--docs) component.

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Button Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/button/).

| ARIA            | Usage                                                                                                             |
| --------------- | ----------------------------------------------------------------------------------------------------------------- |
| `role="button"` |                                                                                                                   |
| `aria-label`    | If no accessible label is provided via the button's text, the `aria-label` or `aria-labelledby` attribute is set. |
| `aria-disabled` | If the button is disabled, the `aria-disabled` attribute is set to `true`.                                        |

### Keyboard

| Key              | Action                |
| ---------------- | --------------------- |
| <kbd>ENTER</kbd> | Activates the button. |
| <kbd>SPACE</kbd> | Activates the button. |
