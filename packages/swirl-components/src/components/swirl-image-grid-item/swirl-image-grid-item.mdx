# SwirlImageGridItem

[SwirlImageGrid](?path=/docs/components-swirlimagegrid--docs) component.

## Accessibility

SwirlImageGridItem is a presentational wrapper for grid images with no ARIA
attributes or keyboard-specific behavior. Provide accessible names for images
via slotted content or parent context.

### Keyboard

No component-specific keyboard behavior.
