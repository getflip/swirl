## Theming

| Variable                        | Usage                                             |
| ------------------------------- | ------------------------------------------------- |
| `--s-text-default`              | Text color of the button                          |
| `--s-icon-default`              | Icon color of the button                          |
| `--s-surface-default`           | Background color of the button                    |
| `--s-surface-hovered`           | Background color of the hovered button            |
| `--s-surface-pressed`           | Background color of the pressed button            |
| `--s-surface-raised-default`    | Background color of the active button             |
| `--s-surface-raised-hovered`    | Background color of the active and hovered button |
| `--s-surface-raised-pressed`    | Background color of the active and pressed button |
| `--s-surface-highlight-default` | Left border color of the active button            |
