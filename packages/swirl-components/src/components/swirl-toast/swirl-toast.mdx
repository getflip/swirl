# FlipToast

The FlipToast component is used to give brief, non-disruptive feedback after an
action has taken place.

## Theming

| Variable                       | Usage                                             |
| ------------------------------ | ------------------------------------------------- |
| `--s-surface-neutral-default`  | Background color for default toast                |
| `--s-surface-critical-default` | Background color for toast with intent "critical" |
| `--s-surface-success-default`  | Background color for toast with intent "success"  |
| `--s-text-on-status`           | Text color for toasts                             |
| `--s-icon-on-status`           | Icon color for toasts                             |
