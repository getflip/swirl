# SwirlToast

The SwirlToast component is used to give brief, non-disruptive feedback after an
action has taken place.

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=2%3A47)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-toast)**

## Usage

<Canvas of={Stories.SwirlToast}></Canvas>

<Source
  code={`<swirl-toast icon="<swirl-icon-mail></swirl-icon-mail>">
    This is toast text with multiple lines. Try to keep the info as short as possible.
</swirl-toast>`}
/>

## Accessibility

Please refer to the
[ToastProvider](?path=/docs/components-swirltoastprovider--docs) component for
information on how to use the SwirlToasts in an accessible way.

### Keyboard

No component-specific keyboard behavior; toasts are announced via the live
region.
