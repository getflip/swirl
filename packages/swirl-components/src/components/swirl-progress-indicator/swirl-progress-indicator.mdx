## Accessibility

### ARIA

| ARIA                 | Usage                                                                                            |
| -------------------- | ------------------------------------------------------------------------------------------------ |
| `role="progressbar"` | Implicit through the `progress` element with `bar` variant; explicitly set for variant `circle`. |
| `aria-label`         | Set to provide an accessible label.                                                              |
| `aria-valuenow`      | Set to the current progress value.                                                               |
| `aria-valuemax`      | Set if max. value is not 100.                                                                    |
| `aria-valuemin`      | Set if min. value is not 0.                                                                      |

### Keyboard

No component-specific keyboard behavior.
