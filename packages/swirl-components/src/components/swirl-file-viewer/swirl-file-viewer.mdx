### Supported file types

| Type            | Example                         |
| --------------- | ------------------------------- |
| image/\*        | image/jpeg<br/>/sample.jpg      |
| video/\*        | video/mp4<br/>/sample.mp4       |
| audio/\*        | audio/mp3<br/>/sample.mp3       |
| text/plain      | text/plain<br/>/sample.txt      |
| text/csv        | text/csv<br/>/sample.csv        |
| application/pdf | application/pdf<br/>/sample.pdf |
