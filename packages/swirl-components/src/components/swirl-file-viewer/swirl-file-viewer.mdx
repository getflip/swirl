### Supported file types

| Type            | Example                         |
| --------------- | ------------------------------- |
| image/\*        | image/jpeg<br/>/sample.jpg      |
| video/\*        | video/mp4<br/>/sample.mp4       |
| audio/\*        | audio/mp3<br/>/sample.mp3       |
| text/plain      | text/plain<br/>/sample.txt      |
| text/csv        | text/csv<br/>/sample.csv        |
| application/pdf | application/pdf<br/>/sample.pdf |

## Accessibility

SwirlFileViewer delegates rendering to format-specific viewers (e.g. PDF, image,
video). Ensure slotted or embedded content provides appropriate labels and
keyboard access; no component-level ARIA table.

### Keyboard

No component-specific keyboard behavior; keyboard access depends on the embedded
viewer.
