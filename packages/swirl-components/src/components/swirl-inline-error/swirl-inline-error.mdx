# SwirlInlineError

The SwirlInlineError component is used to convey information about an invalid
form input.

## Accessibility

### ARIA

| ARIA               | Usage                                                                                                                                                                                             |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `aria-describedby` | The inline error needs to be linked to the corresponding form field by setting the `aria-describedby` attribute on the related field and the matching `id` attribute on the inline error element. |
