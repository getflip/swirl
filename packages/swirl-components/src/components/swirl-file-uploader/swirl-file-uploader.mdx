## Accessibility

### ARIA

The component uses the native HTML file input.

| ARIA            | Usage                                                                     |
| --------------- | ------------------------------------------------------------------------- |
| `aria-disabled` | If the input is disabled, the `aria-disabled` attribute is set to `true`. |

### Keyboard

| Key              | Action                    |
| ---------------- | ------------------------- |
| <kbd>ENTER</kbd> | Activates the file input. |
| <kbd>SPACE</kbd> | Activates the file input. |
