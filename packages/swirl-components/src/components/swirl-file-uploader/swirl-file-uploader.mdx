## Theming

| Variable                          | Usage                                    |
| --------------------------------- | ---------------------------------------- |
| `--s-text-default`                | Text color of the label and description  |
| `--s-interactive-primary-default` | Text color of the dropzone's upload link |
| `--s-icon-highlight`              | Icon color of the dropzone               |
| `--s-border-strong`               | Border color of the dropzone             |

## Accessibility

### ARIA

The component uses the native HTML file input.

| ARIA            | Usage                                                                     |
| --------------- | ------------------------------------------------------------------------- |
| `aria-disabled` | If the input is disabled, the `aria-disabled` attribute is set to `true`. |

### Keyboard

| Key              | Action                    |
| ---------------- | ------------------------- |
| <kbd>ENTER</kbd> | Activates the file input. |
| <kbd>SPACE</kbd> | Activates the file input. |
