# SwirlMenuItem

The SwirlMenuItem component is used to provider user actions inside a
[SwirlMenu component](?path=/docs/components-swirlmenu--docs).

- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-menu-item)**

## Usage

<Canvas of={Stories.SwirlMenuItem}></Canvas>

<Source
  code={`<swirl-menu-item label="Label">

        label="Jane Doe"
        size="l"

## Related components

- **[swirl-menu](?path=/docs/components-swirlmenu--docs)** (parent) — Menu
  container; use `swirl-menu-item` as a direct child of `swirl-menu`. Items can
  contain a nested `swirl-menu` for submenus.

## Accessibility

### ARIA

See [SwirlMenu component](?path=/docs/components-swirlmenu--swirl-menu).

### Keyboard

Keyboard behavior is defined by SwirlMenu (e.g. arrow keys, Enter, Space).
