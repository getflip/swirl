# SwirlEmptyState

The SwirlEmptyState component is used as placeholder content for otherwise empty
pages.

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=826%3A13398)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-empty-state)**

## Accessibility

SwirlEmptyState is a presentational placeholder with no ARIA attributes or
keyboard-specific behavior. Ensure heading and call-to-action slots provide
accessible labels and focus order.
