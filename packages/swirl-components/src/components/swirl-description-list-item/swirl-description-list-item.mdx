# SwirlDescriptionListItem

The SwirlDescriptionListItem component is used to define terms by providing a
short description. It should only be used inside the
[SwirlDescriptionList component](?path=/docs/components-swirldescriptionlist--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=2%3A24)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-description-list-item)**

## Accessibility

SwirlDescriptionListItem renders term and description semantics for use inside
SwirlDescriptionList. No component-specific ARIA or keyboard behavior beyond
native list semantics.

### Keyboard

No component-specific keyboard behavior.
