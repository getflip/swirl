# FlipDescriptionListItem

The FlipDescriptionListItem component is used to define terms by providing a
short description. It should only be used inside the
[FlipDescriptionList component](?path=/docs/components-flipdescriptionlist--flip-description-list).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=2%3A24)**
