## Theming

| Variable                      | Usage                                           |
| ----------------------------- | ----------------------------------------------- |
| `--s-surface-overlay-default` | Background color of the chip                    |
| `--s-surface-overlay-hovered` | Background color of the hovered chip            |
| `--s-surface-overlay-pressed` | Background color of the pressed chip            |
| `--s-surface-raised-hovered`  | Background color of the avatar on pressed chip  |
| `--s-surface-raised-pressed`  | Background color of the avatar on pressed chip  |
| `--s-border-default`          | Border color of the chip                        |
| `--s-border-critical`         | Border color of the chip with intent "critical" |
| `--s-border-success`          | Border color of the chip with intent "success"  |
| `--s-icon-default`            | Icon color of the chip                          |
| `--s-icon-critical`           | Icon color of the chip with intent "critical"   |
| `--s-icon-success`            | Icon color of the chip with intent "success"    |
| `--s-text-subdued`            | Text color of the chip                          |
| `--s-text-critical`           | Text color of the chip with intent "critical"   |
| `--s-text-success`            | Text color of the chip with intent "success"    |

## Accessibility

### ARIA

| ARIA            | Usage                                                               |
| --------------- | ------------------------------------------------------------------- |
| `role="button"` | For interactive chips; achieved by using a native `button` element. |

### Keyboard

| Key              | Action                          |
| ---------------- | ------------------------------- |
| <kbd>ENTER</kbd> | Activates the interactive chip. |
| <kbd>SPACE</kbd> | Activates the interactive chip. |
