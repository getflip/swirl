## Accessibility

### ARIA

The component follows the
[WAI-ARIA Tree View Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/treeview/).

### Keyboard

Keyboard behavior is defined by the tree view (e.g. arrow keys, Enter, Space for
expand/collapse and selection).
