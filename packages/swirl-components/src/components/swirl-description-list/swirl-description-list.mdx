# SwirlDescriptionList

The SwirlDescriptionList component is used to provide a list of terms with short
descriptions. It is used in conjunction with the
[SwirlDescriptionListItem component](?path=/docs/components-swirldescriptionlistitem--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=2%3A24)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-description-list)**

## Accessibility

The component uses a HTML description list to communicate the relationship
between the terms and descriptions to assistive technology.

### Keyboard

No component-specific keyboard behavior.
