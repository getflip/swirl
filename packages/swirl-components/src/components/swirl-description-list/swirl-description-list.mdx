# SwirlDescriptionList

The SwirlDescriptionList component is used to provide a list of terms with short
descriptions. It is used in conjunction with the
[SwirlDescriptionListItem component](?path=/docs/components-swirldescriptionlistitem--swirl-description-list-item).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=2%3A24)**
