## Accessibility

### ARIA

The component follows the
[WAI-ARIA Carousel Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/carousel/).

| ARIA                   | Usage                       |
| ---------------------- | --------------------------- |
| `role="group"`         |                             |
| `aria-roledescription` | Set to "slide".             |
| `aria-label`           | Set on the slide container. |
