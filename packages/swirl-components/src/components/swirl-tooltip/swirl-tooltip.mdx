## Accessibility

### ARIA

The component follows the
[WAI-ARIA Tooltip Widget Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/tooltip/)
(WIP).

| ARIA               | Usage                                                                                                        |
| ------------------ | ------------------------------------------------------------------------------------------------------------ |
| `role="tooltip"`   | The tooltip container has the `tooltip` role.                                                                |
| `aria-describedby` | The element that triggers the tooltip gets the `aria-describedby` attribute referencing the tooltip element. |

### Keyboard

| Key            | Action                 |
| -------------- | ---------------------- |
| <kbd>ESC</kbd> | Dismisses the tooltip. |
