## Accessibility

### ARIA

The component follows the
[WAI-ARIA Alert and Message Dialogs Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/alertdialog/).

| ARIA                 | Usage                                               |
| -------------------- | --------------------------------------------------- |
| `role="alertdialog"` | The dialog has the `alertdialog` role.              |
| `aria-hidden`        | Set to "true" if dialog is hidden.                  |
| `aria-modal`         | Set to "true".                                      |
| `aria-label`         | Set on the dialog if label is hidden.               |
| `aria-labelledby`    | Set on the dialog to refer to the dialog's label.   |
| `aria-describedby`   | Set on the dialog to refer to the dialog's content. |

### Keyboard

| Key                                               | Action                                                                                                  |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| <kbd>ESC</kbd>                                    | Closes the dialog.                                                                                      |
| <kbd>TAB</kbd>, <kbd>SHIFT</kbd> + <kbd>TAB</kbd> | Cycles through the focusable elements inside the dialog. Focus is trapped inside the dialog while open. |

### Focus handling

- Focus is moved to the secondary button when the dialog is opened. If no
  secondary button is provided, the primary button is focused.
- After closing, the focus is returned to last the focused element.
