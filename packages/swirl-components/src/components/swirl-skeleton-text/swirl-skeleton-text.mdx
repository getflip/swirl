# SwirlSkeletonText

The SwirlSkeletonText component is used to display a low fidelity representation
of text before it is loaded.

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Swirl-Components?node-id=2929%3A17884&t=Que3BOUIEmzL9qsn-0)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-skeleton-text)**
