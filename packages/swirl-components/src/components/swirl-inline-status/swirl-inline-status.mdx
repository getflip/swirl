## Theming

| Variable            | Usage                                                  |
| ------------------- | ------------------------------------------------------ |
| `--s-icon-critical` | Icon color of the inline status with intent "critical" |
| `--s-text-critical` | Text color of the inline status with intent "critical" |
| `--s-icon-info`     | Icon color of the inline status with intent "info"     |
| `--s-text-info`     | Text color of the inline status with intent "info"     |
| `--s-icon-success`  | Icon color of the inline status with intent "success"  |
| `--s-text-success`  | Text color of the inline status with intent "success"  |
| `--s-icon-warning`  | Icon color of the inline status with intent "warning"  |
| `--s-text-warning`  | Text color of the inline status with intent "warning"  |

## Accessibility

### ARIA

| ARIA               | Usage                                                                                                                                                                                              |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `aria-describedby` | The inline status needs to be linked to the corresponding form field by setting the `aria-describedby` attribute on the related field and the matching `id` attribute on the inline error element. |
