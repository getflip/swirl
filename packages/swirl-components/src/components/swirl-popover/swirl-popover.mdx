## Accessibility

### ARIA

| ARIA            | Usage                                                   |
| --------------- | ------------------------------------------------------- |
| `role="dialog"` | The popover content has `role` set to `dialog`.         |
| `aria-label`    | Describes the popover content.                          |
| `aria-controls` | Associates the popover and its trigger.                 |
| `aria-expanded` | Used on the trigger to convey the state of the popover. |
| `aria-haspopup` | Set on the trigger to denote that it opens a popover.   |

### Keyboard

| Key              | Action                                       |
| ---------------- | -------------------------------------------- |
| <kbd>ENTER</kbd> | Activates the trigger and opens the popover. |
| <kbd>SPACE</kbd> | Activates the trigger and opens the popover. |
| <kbd>ESC</kbd>   | Closes the popover.                          |

### Focus handling

- Focus is moved to the popover content when opened.
- After closing, the focus is returned to the trigger.
- Moving focus outside the popover closes it.
