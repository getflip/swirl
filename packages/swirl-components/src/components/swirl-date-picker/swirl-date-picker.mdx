## Accessibility

SwirlDatePicker is typically used inside SwirlDateInput or a popover; its calendar grid and navigation are exposed for assistive technologies by the host. No component-specific ARIA table or keyboard section here; refer to the embedding component (e.g. SwirlDateInput) for full behavior.

### Keyboard

Calendar navigation and date selection are handled by the embedding component or host.
