## Accessibility

### ARIA

The component uses buttons for interactive elements with appropriate titles for
screen readers.

| ARIA           | Usage                                                                       |
| -------------- | --------------------------------------------------------------------------- |
| `role="group"` | Set on the chip container                                                   |
| `title`        | The file name is set as a title on the name element for tooltip on overflow |
| Button titles  | The preview and download buttons have titles for screen readers             |

### Keyboard

| Key              | Action                                |
| ---------------- | ------------------------------------- |
| <kbd>ENTER</kbd> | Activates the preview/download button |
| <kbd>SPACE</kbd> | Activates the preview/download button |
