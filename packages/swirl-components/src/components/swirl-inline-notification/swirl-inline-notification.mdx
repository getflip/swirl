## Accessibility

### ARIA

The component follows the
[WAI-ARIA Alert Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/alert/).

| ARIA            | Usage                                                                      |
| --------------- | -------------------------------------------------------------------------- |
| `role="alert"`  | The banner gets the `alert` role if it contains critical info or warnings. |
| `role="status"` | The banner gets the `status` role for non-critical info.                   |
