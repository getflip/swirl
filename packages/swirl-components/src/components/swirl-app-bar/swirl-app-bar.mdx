# SwirlAppBar

The SwirlAppBar component is used to display a header app bar inside the
[SwirlAppLayout](?path=/docs/components-swirlapplayout--docs) or
[SwirlModal](?path=/docs/components-swirlmodal--docs).

- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-app-bar)**

## Accessibility

SwirlAppBar is a layout container for header content. Buttons (back, close,
stepper) use accessible labels via props. No component-level ARIA role or
keyboard shortcuts; interaction is handled by slotted controls.

### Keyboard

No component-specific keyboard behavior; slotted buttons and links handle focus
and activation.
