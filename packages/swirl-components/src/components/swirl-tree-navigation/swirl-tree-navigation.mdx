## Accessibility

### ARIA

The component follows the [WAI-ARIA Tree View Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/treeview/). The container is a tree; keyboard and item semantics are on [SwirlTreeNavigationItem](?path=/docs/admin-swirltreenavigationitem--docs).

| ARIA           | Usage                                                         |
| -------------- | ------------------------------------------------------------- |
| `role="tree"`  | Set on the root list container.                               |
| `aria-label`   | Set via the `label` prop to name the tree.                    |

### Keyboard

Keyboard interaction is handled by the tree items; see [SwirlTreeNavigationItem](?path=/docs/admin-swirltreenavigationitem--docs).
