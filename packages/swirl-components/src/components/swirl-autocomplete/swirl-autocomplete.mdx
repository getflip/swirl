# SwirlAutocomplete

The SwirlAutocomplete component is used to provide a text input field showing
selectable suggestions while the user interacts with the input. It should always
be used in combination with the
[SwirlFormControl component](?path=/docs/components-swirlformcontrol--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Swirl-Components?node-id=534%3A8927)**

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Combobox Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/combobox/).

| ARIA                       | Usage                                                  |
| -------------------------- | ------------------------------------------------------ |
| `role="combobox"`          | Set on the text input.                                 |
| `aria-autocomplete="list"` | Set on the text input.                                 |
| `aria-controls`            | Set on the text input. References the suggestion menu. |
| `aria-expanded`            | Set on the text input.                                 |

The suggestion menu uses the
[SwirlOptionList component](?path=/docs/components-swirloptionlist--docs).

### Keyboard

| Key                                | Action                                                         |
| ---------------------------------- | -------------------------------------------------------------- |
| <kbd>DOWN</kbd>                    | Moves focus to the suggestion menu.                            |
| <kbd>ESC</kbd>                     | Closes the suggestion menu.                                    |
| <kbd>SPACE</kbd>, <kbd>ENTER</kbd> | Sets the value to the selected suggestion and closes the menu. |

See the
[SwirlOptionList component](?path=/docs/components-swirloptionlist--docs) for
further keyboard controls.
