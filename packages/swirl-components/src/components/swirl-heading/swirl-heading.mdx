## Accessibility

Please refer to the
["Headings" part](https://www.w3.org/WAI/tutorials/page-structure/headings/) of
the
[W3C's Page Structure Tutorial](https://www.w3.org/WAI/tutorials/page-structure/)
on how to use headings correctly. The component renders a native heading element
(`h1`–`h6`) via the `as` prop; no component-specific ARIA or keyboard behavior.
