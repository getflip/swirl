## Accessibility

| ARIA          | Usage                                                                                          |
| ------------- | ---------------------------------------------------------------------------------------------- |
| `role="img"`  | Set on the SVG when a `label` is provided, with the label text exposed via an inner `<title>`. |
| `aria-hidden` | Set to `true` automatically when no `label` is provided so the icon is treated as decorative.  |
