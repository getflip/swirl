# SwirlAvatarGroup

The SwirlAvatarGroup component is used to display a stack of two avatars.

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=2%3A12)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-avatar-group)**

<Canvas of={Stories.SwirlAvatarGroup}></Canvas>

<Source
  code={`<swirl-avatar-group badge='<swirl-badge aria-label="3 new messages" label="3"></swirl-badge>'>

## Accessibility

### ARIA

| ARIA           | Usage                                                                          |
| -------------- | ------------------------------------------------------------------------------ |
| `role="group"` | Default role for the avatar-group                                              |
| `role="list"`  | Configurable role for the avatar-group. Children should have `role="listitem"` |

No component-specific keyboard behavior.
