# SwirlPopoverTrigger

The SwirlPopoverTrigger component is used to position and open a referenced
[SwirlPopover component](?path=/docs/components-swirlpopover--docs).

- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-popover-trigger)**

## Usage

<Canvas of={Stories.SwirlPopoverTrigger} sourceState="shown"></Canvas>

<Controls of={Stories.SwirlPopoverTrigger} />

ℹ️ **Note:** Referencing a popover via ID only works when the popover is part of
the light DOM. Pass the popover DOM element directly, if the popover is behind a
shadow DOM barrier.

## Accessibility

### ARIA

| ARIA            | Usage                                                                |
| --------------- | -------------------------------------------------------------------- |
| `aria-controls` | Set on the wrapped trigger element, referencing the popover element. |
| `aria-expanded` | Set on the wrapped trigger element.                                  |
| `aria-haspopup` | Set to "dialog" on the wrapped trigger element.                      |
