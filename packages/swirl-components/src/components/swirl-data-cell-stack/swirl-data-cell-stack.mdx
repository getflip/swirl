# SwirlDataCellStack

The SwirlDataCellStack component is a container for grouping multiple
SwirlDataCell components with an optional label, description, and call-to-action
slot.

- **[Figma Design Specs](https://www.figma.com/design/aIsxMzrP5myDMpvxaRIHXB/Web-Components-2.0?node-id=147-680)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-data-cell-stack)**

## Properties

| Property      | Attribute     | Description                              | Type      | Default     |
| ------------- | ------------- | ---------------------------------------- | --------- | ----------- |
| `description` | `description` | Optional description text                | `string`  | `undefined` |
| `hideLabel`   | `hide-label`  | Hide the label visually (default: false) | `boolean` | `false`     |
| `label`       | `label`       | Optional label text                      | `string`  | `undefined` |

## Slots

| Slot      | Description                             |
| --------- | --------------------------------------- |
| `cta`     | Optional call-to-action (e.g., button)  |
| (default) | The SwirlDataCell components to display |

## Related components

- **[swirl-data-cell](?path=/docs/components-swirldatacell--docs)** (child) —
  Data cells; use as direct children in the default slot of
  `swirl-data-cell-stack`.

## Accessibility

### ARIA

| ARIA               | Usage                                             |
| ------------------ | ------------------------------------------------- |
| `role="group"`     | Set on the stack container.                       |
| `role="list"`      | Set on the list of data cells.                    |
| `aria-labelledby`  | References the optional label when present.       |
| `aria-describedby` | References the optional description when present. |

### Keyboard

No component-specific keyboard behavior; interaction is handled by slotted
SwirlDataCell content.
