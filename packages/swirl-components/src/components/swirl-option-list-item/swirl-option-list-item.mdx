# SwirlOptionListItem

The SwirlOptionListItem component is used to provide a user-selectable option.
It should only be used inside the
[SwirlOptionList component](?path=/docs/components-swirloptionlist--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=448%3A6840)**

## Theming

| Variable                      | Usage                                |
| ----------------------------- | ------------------------------------ |
| `--s-surface-overlay-default` | Background color of the item         |
| `--s-surface-overlay-hovered` | Background color of the hovered item |
| `--s-surface-overlay-pressed` | Background color of the active item  |
| `--s-text-default`            | Text color of the item               |
| `--s-text-disabled`           | Text color of the disabled item      |
| `--s-text-highlight`          | Text color of the selected item      |
| `--s-icon-default`            | Icon color of the item               |
| `--s-icon-highlight`          | Icon color of the selected item      |

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Listbox Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/listbox/).

| ARIA                   | Usage                        |
| ---------------------- | ---------------------------- |
| `role="option"`        |                              |
| `aria-disabled="true"` | Set if the item is disabled. |
| `aria-selected="true"` | Set if the item is selected. |
