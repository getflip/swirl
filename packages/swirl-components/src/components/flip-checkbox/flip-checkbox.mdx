# FlipCheckbox

The FlipCheckbox component is used as a form control to allow a user to select
one or multiple choices from a list of options.

## Theming

| Variable                 | Usage                                                                                                      |
| ------------------------ | ---------------------------------------------------------------------------------------------------------- |
| `--s-icon-default`       | Border color of the unchecked checkbox                                                                     |
| `--s-icon-highlight`     | Border and background color of checked and indeterminate checkbox                                          |
| `--s-background-hovered` | Backdrop color of hovered checkbox                                                                         |
| `--s-background-pressed` | Backdrop color of pressed checkbox                                                                         |
| `--s-icon-disabled`      | Border color of disabled, unchecked checkbox; background color of disabled, checked/indeterminate checkbox |
| `--s-text-disabled`      | Text color of disabled checkbox                                                                            |
| `--s-focus`              | Outline color of focused checkbox                                                                          |

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Checkbox Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/checkbox/).

| ARIA              | Usage                                                                                                                                                |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| `role="checkbox"` |                                                                                                                                                      |
| `aria-label`      | If no accessible label is provided via the checkbox's label prop, the `aria-label` or `aria-labelledby` attribute has to be set.                     |
| `aria-checked`    | The checkbox gets the `aria-checked` attribute set to `true` in checked state, to `false` in unchecked state, and to `mixed` in indeterminate state. |

### Keyboard

| Key              | Action                             |
| ---------------- | ---------------------------------- |
| <kbd>SPACE</kbd> | Changes the state of the checkbox. |
