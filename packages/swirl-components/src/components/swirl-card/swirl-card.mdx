## Accessibility

### ARIA

When used as a link (`href`) or with `swirlAriaLabel` / `swirlAriaLabelledby`, the card exposes optional ARIA for assistive technologies.

| ARIA               | Usage                                                                 |
| ------------------ | --------------------------------------------------------------------- |
| `aria-current`     | Optional; set via `swirlAriaCurrent` (e.g. `"page"`, `"step"`).       |
| `aria-label`       | Optional; set via `swirlAriaLabel` when the card has no visible text. |
| `aria-labelledby`  | Optional; set via `swirlAriaLabelledby` to reference a visible label. |

### Keyboard

When the card is interactive (e.g. has `href`), it uses native link semantics: <kbd>Enter</kbd> activates. No component-specific shortcuts.
