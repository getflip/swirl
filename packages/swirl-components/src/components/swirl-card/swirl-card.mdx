## Theming

| Variable               | Usage                                            |
| ---------------------- | ------------------------------------------------ |
| `--s-surface-default`  | Background color of the card                     |
| `--s-surface-hovered`  | Background color of the hovered interactive card |
| `--s-surface-pressed`  | Background color of the active interactive card  |
| `--s-border-default`   | Border color of the card                         |
| `--s-border-highlight` | Border color of the highlighted card             |
