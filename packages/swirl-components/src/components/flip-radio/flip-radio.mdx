# FlipRadio

The FlipRadio component is used as a form control to allow a user to select a
single choice from a list of options. Has to be used in combination with a
[FlipRadioGroup](?path=/docs/components-flipradiogroup--flip-radio-group).

- **[Figma Design Specs](https://www.figma.com/file/l498Zs9zdZ1qrFJ8aYs47G/Components-V2?node-id=29%3A643)**

## Theming

| Variable                 | Usage                                                                                                |
| ------------------------ | ---------------------------------------------------------------------------------------------------- |
| `--s-icon-default`       | Border color of the unchecked radio button                                                           |
| `--s-icon-highlight`     | Border and background color of the checked radio button                                              |
| `--s-background-hovered` | Backdrop color of hovered radio button                                                               |
| `--s-background-pressed` | Backdrop color of pressed radio button                                                               |
| `--s-icon-disabled`      | Border color of disabled, unchecked radio button; background color of disabled, checked radio button |
| `--s-text-disabled`      | Text color of disabled radio button                                                                  |
| `--s-focus`              | Outline color of focused radio button                                                                |

## Accessibility

All radio buttons have to be contained in a
[FlipRadioGroup](?path=/docs/components-flipradiogroup--flip-radio-group).

### ARIA

| ARIA           | Usage                                                                                                                         |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| `role="radio"` |                                                                                                                               |
| `aria-label`   | If no accessible label is provided via the radio's label prop, the `aria-label` or `aria-labelledby` attribute has to be set. |
| `aria-checked` | The radio button gets the `aria-checked` attribute set to `true` in checked state, to `false` in unchecked state.             |

### Keyboard

| Key              | Action                      |
| ---------------- | --------------------------- |
| <kbd>SPACE</kbd> | Activates the radio button. |
