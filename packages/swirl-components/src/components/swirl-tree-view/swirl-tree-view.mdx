# SwirlTreeView

The SwirlTreeView component is used to render an interactive, nested and
hierarchical list of items.

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Tree View Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/treeview/).

### Keyboard

Keyboard behavior is defined by the tree view host and items (e.g. arrow keys,
Enter, Space).
