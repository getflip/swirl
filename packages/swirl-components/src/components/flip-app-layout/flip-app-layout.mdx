# FlipAppLayout

The FlipAppLayout component provides a basic layout used by app integrations to
arrange content on the screen.

- **[Figma Design Specs](#)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/flip-app-layout)**
