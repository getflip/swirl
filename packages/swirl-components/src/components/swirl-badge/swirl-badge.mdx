## Accessibility

### ARIA

| ARIA            | Usage                                                                                                                                       |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |
| `role="status"` | All badges have the aria role "status" added.                                                                                               |
| `aria-label`    | If no accessible label is provided via the label prop (e.g. a plain number), the `aria-label` or `aria-labelledby` attribute has to be set. |
