# SwirlSelect

The SwirlSelect component is used to allow users to select one or multiple
options from a predefined list. **It's value is always an array of type string even though it is a single select**. It should always be used in combination with the
[SwirlFormControl component](?path=/docs/components-swirlformcontrol--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Swirl-Components?node-id=2%3A43)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-select)**

## Theming

| Variable            | Usage                                    |
| ------------------- | ---------------------------------------- |
| `--s-text-default`  | Text color of the select value           |
| `--s-text-disabled` | Text color of the disabled select value  |
| `--s-icon-default`  | Icon color of the select toggle          |
| `--s-icon-disabled` | Icon color of the disabled select toggle |
