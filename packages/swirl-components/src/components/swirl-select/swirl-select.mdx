# SwirlSelect

The SwirlSelect component is used to allow users to select one or multiple
options from a predefined list. **It's value is always an array of type string even though it is a single select**. It should always be used in combination with the
[SwirlFormControl component](?path=/docs/components-swirlformcontrol--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Swirl-Components?node-id=2%3A43)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-select)**

## Related components

- **[swirl-form-control](?path=/docs/components-swirlformcontrol--docs)** (parent) — Wraps the select with label and description; use `swirl-select` in the default slot of `swirl-form-control`.
- **[swirl-option-list](?path=/docs/components-swirloptionlist--docs)** (internal) — Select uses `swirl-option-list` and [swirl-popover](?path=/docs/components-swirlpopover--docs) for the dropdown.

## Accessibility

See the documentation of the
[SwirlOptionList](?path=/docs/components-swirloptionlist--docs) and
[SwirlPopover](?path=/docs/components-swirlpopover--docs) components.

### Keyboard

Keyboard behavior is defined by SwirlOptionList and SwirlPopover (e.g. arrow keys, Enter, Escape).
