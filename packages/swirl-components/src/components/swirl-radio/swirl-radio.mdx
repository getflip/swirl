# SwirlRadio

The SwirlRadio component is used as a form control to allow a user to select a
single choice from a list of options. Has to be used in combination with a
[SwirlRadioGroup](?path=/docs/components-swirlradiogroup--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=2%3A41)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-radio)**

## Accessibility

All radio buttons have to be contained in a
[SwirlRadioGroup](?path=/docs/components-swirlradiogroup--docs).

### ARIA

| ARIA           | Usage                                                                                                                         |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| `role="radio"` |                                                                                                                               |
| `aria-label`   | If no accessible label is provided via the radio's label prop, the `aria-label` or `aria-labelledby` attribute has to be set. |
| `aria-checked` | The radio button gets the `aria-checked` attribute set to `true` in checked state, to `false` in unchecked state.             |

### Keyboard

| Key              | Action                      |
| ---------------- | --------------------------- |
| <kbd>SPACE</kbd> | Activates the radio button. |
