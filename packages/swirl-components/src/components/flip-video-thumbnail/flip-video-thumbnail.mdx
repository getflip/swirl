# FlipVideoThumbnail

The FlipVideoThumbnail component is used to present a clickable video preview
with optional information about the video's duration.

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Swirl-Components?node-id=205%3A4338)**

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Button Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/button/).

| ARIA               | Usage                               |
| ------------------ | ----------------------------------- |
| `role="button"`    |                                     |
| `aria-label`       | Used to assign an accessible label. |
| `aria-describedby` | Used to associate the duration.     |

### Keyboard

| Key              | Action                   |
| ---------------- | ------------------------ |
| <kbd>ENTER</kbd> | Activates the thumbnail. |
| <kbd>SPACE</kbd> | Activates the thumbnail. |
