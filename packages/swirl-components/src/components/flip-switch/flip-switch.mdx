# FlipSwitch

The FlipSwitch component is used as an alternative to the FlipCheckbox, if an
on/off state is more appropriate than a checked/unchecked state.

- **[Figma Design Specs](https://www.figma.com/file/l498Zs9zdZ1qrFJ8aYs47G/Components-V2?node-id=2127%3A21002)**

## Theming

| ---------------------------------- | --------------------------------------------- |
| `--s-surface-default`              | Background color of the thumb                 |
| `--s-interactive-neutral-default`  | Background color of the disabled thumb        |

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Switch Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/switch/).

| ARIA            | Usage                                                                                                                          |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `role="switch"` |                                                                                                                                |
| `aria-label`    | If no accessible label is provided via the switch's label prop, the `aria-label` or `aria-labelledby` attribute has to be set. |
| `aria-checked`  | The switch gets the `aria-checked` attribute set to `true` in "on" state, and to `false` in "off" state.                       |

### Keyboard

| Key              | Action                           |
| ---------------- | -------------------------------- |
| <kbd>SPACE</kbd> | Changes the state of the switch. |
