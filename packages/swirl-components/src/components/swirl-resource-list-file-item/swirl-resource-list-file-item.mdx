# SwirlResourceListFileItem

The SwirlResourceListFileItem component can be used to represent a file upload
within a collection. It should only be used inside the
[SwirlResourceList component](?path=/docs/components-swirlresourcelist--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=2%3A28)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-resource-list-file-item)**

## Accessibility

### ARIA

| ARIA              | Usage                                                                                 |
| ----------------- | ------------------------------------------------------------------------------------- |
| `role="row"`      | Set on the element; for usage inside the SwirlResourceList component.                 |
| `role="gridcell"` | Set on the inner container element; for usage inside the SwirlResourceList component. |

### Keyboard

| Key              | Action                                                                                  |
| ---------------- | --------------------------------------------------------------------------------------- |
| <kbd>ENTER</kbd> | Activates the item if used as a button or link.                                         |
| <kbd>SPACE</kbd> | Activates the item if used as a button or link. Toggles the item if used as a checkbox. |
