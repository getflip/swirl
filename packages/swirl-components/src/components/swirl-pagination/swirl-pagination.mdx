## Accessibility

### ARIA

The component uses a native HTML `nav` element. A label has to be provided to
make the pagination distinguishable from other navigation blocks and the app's
main navigation.

| ARIA                  | Usage                                                  |
| --------------------- | ------------------------------------------------------ |
| `role="navigation"`   | Implicit through the use of the `nav` element.         |
| `aria-current="page"` | Set on the current page label.                         |
| `aria-label`          | Set to provide an accessible label for the pagination. |
