# FlipToast

The FlipToast component is used to give brief, non-disruptive feedback after an
action has taken place.

## Theming

| Variable                       | Usage                                                  |
| ------------------------------ | ------------------------------------------------------ |
| `--s-surface-neutral-default`  | Background color for default toast                     |
| `--s-surface-critical-default` | Background color for toast with intent "critical"      |
| `--s-surface-warning-default`  | Background color for toast with intent "warning"       |
| `--s-surface-success-default`  | Background color for toast with intent "success"       |
| `--s-surface-info-default`     | Background color for toast with intent "info"          |
| `--s-text-on-status`           | Text color for toast with intents other than "warning" |
| `--s-text-default`             | Text color for toast with intent "warning"             |
| `--s-icon-on-status`           | Icon color for toast with intents other than "warning" |
| `--s-icon-strong`              | Icon color for toast with intent "warning"             |
