# SwirlToggleButton

The SwirlToggleButton component is used to allow a user to select a single
option from a group of options. Once selected, the option can only be unselected
by selecting a different one. The `identifier` has to be provided so the
SwirlToggleGroup component can manage the pressed state of the options. Has to
be used in combination with a
[SwirlToggleGroup](?path=/docs/components-swirltogglegroup--docs).

- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-toggle-button)**

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Button Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/button/).

| ARIA            | Usage                                                                     |
| --------------- | ------------------------------------------------------------------------- |
| `role="button"` |                                                                           |
| `aria-pressed`  | If the button is selected, the `aria-pressed` attribute is set to `true`. |

### Keyboard

| Key              | Action              |
| ---------------- | ------------------- |
| <kbd>ENTER</kbd> | Toggles the button. |
| <kbd>SPACE</kbd> | Toggles the button. |
