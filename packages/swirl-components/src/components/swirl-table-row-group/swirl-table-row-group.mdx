## Accessibility

### ARIA

The component follows the
[WAI-ARIA Table Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/table/).

| ARIA               | Usage                                      |
| ------------------ | ------------------------------------------ |
| `role="rowgroup"`  |                                            |
| `role="row"`       | Set on a the groups row element.           |
| `role="rowheader"` | Set on the header cell element.            |
| `aria-colspan`     | Set to the amount of columns of the table. |
