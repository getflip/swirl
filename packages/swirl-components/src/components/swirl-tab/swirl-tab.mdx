# SwirlTab

The SwirlTab component is used to display a tab panel inside the
[SwirlTabs component](?path=/docs/components-swirltabs--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=483%3A9675)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-tab)**

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Tabs Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/tabpanel/).

| ARIA              | Usage                                                        |
| ----------------- | ------------------------------------------------------------ |
| `role="tabpanel"` |                                                              |
| `aria-labelledby` | Set on each tab panel to associate it with its tab bar item. |

### Keyboard

Keyboard behavior is defined by SwirlTabs / SwirlTabBar (e.g. arrow keys for tab
focus).
