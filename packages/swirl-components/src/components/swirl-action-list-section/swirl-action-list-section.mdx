## Theming

| Variable                      | Usage                           |
| ----------------------------- | ------------------------------- |
| `--s-surface-overlay-default` | Background color of the section |

## Accessibility

### ARIA

| ARIA           | Usage                                      |
| -------------- | ------------------------------------------ |
