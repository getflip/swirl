# SwirlTimeInput

The SwirlDateInput component is used to enter time values. It should always be
used in combination with the
[SwirlFormControl component](?path=/docs/components-swirlformcontrol--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Swirl-Components?type=design&node-id=205-4334&t=oPzrbpq03fbccFIJ-0)**

## Accessibility

### ARIA

| ARIA               | Usage                                     |
| ------------------ | ----------------------------------------- |
| `aria-describedby` | Optional; set via `swirlAriaDescribedby`. |
| `aria-disabled`    | Set when the input is disabled.           |
| `aria-invalid`     | Set when the value is invalid.            |

### Keyboard

Standard text input behavior. No component-specific shortcuts.
