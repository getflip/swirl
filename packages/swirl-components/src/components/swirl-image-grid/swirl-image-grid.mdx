# SwirlImageGrid

The SwirlImageGrid component is used to display a variable amount of images
inside a grid.

## Accessibility

SwirlImageGrid is a presentational grid container with no ARIA attributes or
keyboard-specific behavior. Ensure SwirlImageGridItem or slotted images have
accessible labels where needed.

### Keyboard

No component-specific keyboard behavior.
