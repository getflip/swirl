## Theming

| Variable                        | Usage                                           |
| ------------------------------- | ----------------------------------------------- |
| `--s-text-default`              | Text color of the item                          |
| `--s-icon-default`              | Icon color of the item                          |
| `--s-surface-default`           | Background color of the item                    |
| `--s-surface-hovered`           | Background color of the hovered item            |
| `--s-surface-pressed`           | Background color of the pressed item            |
| `--s-surface-raised-default`    | Background color of the active item             |
| `--s-surface-raised-hovered`    | Background color of the active and hovered item |
| `--s-surface-raised-pressed`    | Background color of the active and pressed item |
| `--s-surface-highlight-default` | Left border color of the active item            |
