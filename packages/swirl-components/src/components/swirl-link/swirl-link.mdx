## Theming

| Variable                           | Usage                           |
| ---------------------------------- | ------------------------------- |
| `--s-interactive-primary-default`  | Text color of the link          |
| `--s-interactive-primary-hovered`  | Text color of the hovered link  |
| `--s-interactive-primary-pressed`  | Text color of the active link   |
| `--s-interactive-primary-disabled` | Text color of the disabled link |

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Link Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/link/).

| ARIA          | Usage                                         |
| ------------- | --------------------------------------------- |
| `role="link"` | Implicit through usage of the native `a` tag. |

### Keyboard

| Key              | Action                                                |
| ---------------- | ----------------------------------------------------- |
| <kbd>ENTER</kbd> | Executes the link and moves focus to the link target. |
