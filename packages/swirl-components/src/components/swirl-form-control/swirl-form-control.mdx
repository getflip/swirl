# SwirlFormControl

The SwirlFormControl component is used to wrap form elements and provide context
information via labels, descriptions and error messages.

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=205%3A4334)**

## Theming

| Variable               | Usage                                            |
| ---------------------- | ------------------------------------------------ |
| `--s-border-strong`    | Border color of the control                      |
| `--s-border-default`   | Border color of the disabled control             |
| `--s-border-highlight` | Border color of the active control               |
| `--s-border-critical`  | Border color of the invalid control              |
| `--s-text-subdued`     | Text color of the label and description          |
| `--s-text-disabled`    | Text color of the disabled label and description |

## Accessibility

### ARIA

| ARIA               | Usage                                                           |
| ------------------ | --------------------------------------------------------------- |
| `role="group"`     |                                                                 |
| `aria-describedby` | Added to the form element to associate it with the description. |
