# FlipDateInput

The FlipDateInput component is used to render a date input with a date picker.
It should always be used in combination with the
[FlipFormControl component](?path=/docs/components-flipformcontrol--flip-form-control).

## Theming

| Variable               | Usage                                  |
| ---------------------- | -------------------------------------- |
| `--s-text-default`     | Text color of the input value          |
| `--s-text-disabled`    | Text color of the disabled input value |
| `--s-icon-default`     | Icon color of the date picker button   |
| `--s-border-highlight` | Caret color                            |
