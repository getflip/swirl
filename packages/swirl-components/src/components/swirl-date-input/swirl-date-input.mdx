# SwirlDateInput

The SwirlDateInput component is used to render a date input with a date picker.
It should always be used in combination with the
[SwirlFormControl component](?path=/docs/components-swirlformcontrol--docs).

## Accessibility

### ARIA

The input and date picker trigger expose ARIA for form and button semantics.

| ARIA               | Usage                                                                 |
| ------------------ | --------------------------------------------------------------------- |
| `aria-describedby` | Optional; set via `swirlAriaDescribedby`.                             |
| `aria-disabled`    | Set when the control is disabled.                                    |
| `aria-invalid`     | Set when the value is invalid.                                        |
| `aria-label`       | Set on the date picker trigger button via `datePickerTriggerLabel`.   |

### Keyboard

Standard text input and button behavior. The date picker trigger is activated with <kbd>Enter</kbd> or <kbd>Space</kbd>. No component-specific shortcuts.
