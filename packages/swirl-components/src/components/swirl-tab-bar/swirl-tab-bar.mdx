# SwirlTabBar

The SwirlTabBar component is used to render the tabs navigation of the
[SwirlTabs](?path=/docs/components-swirltabs--docs) component. You can use it
standalone in combination with the
[SwirlTab](?path=/docs/components-swirltab--docs) components, if you want to
implement your own tab handling.

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Swirl-Components?node-id=34-6419&t=l6r0AzmK9ThWIT5N-0)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-tab-bar)**

## Related components

- **[swirl-tabs](?path=/docs/components-swirltabs--docs)** (parent) — Tab
  container; `swirl-tab-bar` is typically used inside `swirl-tabs` to render the
  tab strip.
- **[swirl-tab](?path=/docs/components-swirltab--docs)** (sibling) — Tab panels;
  use alongside `swirl-tab-bar` when building a tabbed UI.

## Accessibility

See [SwirlTabs](?path=/docs/components-swirltabs--docs).
