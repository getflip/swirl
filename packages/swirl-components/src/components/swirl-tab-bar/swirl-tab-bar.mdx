# SwirlTabBar

The SwirlTabBar component is used to render the tabs navigation of the
[SwirlTabs](?path=/docs/components-swirltabs--swirl-tabs) component. You can use
it standalone in combination with the
[SwirlTab](?path=/docs/components-swirltab--swirl-tab) components, if you want
to implement your own tab handling.

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Swirl-Components?node-id=34-6419&t=l6r0AzmK9ThWIT5N-0)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-tab-bar)**

## Theming

| Variable                           | Usage                                               |
| ---------------------------------- | --------------------------------------------------- |
| `--s-interactive-neutral-default`  | Text color of inactive tabs                         |
| `--s-interactive-neutral-pressed`  | Text color of active tabs                           |
| `--s-interactive-neutral-disabled` | Text color of disabled tabs                         |
| `--s-text-highlight`               | Text color of the active tab                        |
| `--s-border-highlight`             | Border color of the active tab                      |
| `--s-border-default`               | Border color of the tab bar and hovered/active tabs |
