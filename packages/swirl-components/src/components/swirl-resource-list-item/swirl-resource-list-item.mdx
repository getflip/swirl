# SwirlResourceListItem

The SwirlResourceListItem component can be used to represent an object, like
media, a group or a chat within a collection. It should only be used inside the
[SwirlResourceList component](?path=/docs/components-swirlresourcelist--docs).

- **[Figma Design Specs](https://www.figma.com/file/iRHiAFsHxnzux8gQ9EjlZe/Flip-Components?node-id=236%3A5240)**
- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-resource-list-item)**

## Accessibility

### ARIA

Depending on the purpose, the item can be used as a button, link or checkbox
control.

| ARIA              | Usage                                                                                           |
| ----------------- | ----------------------------------------------------------------------------------------------- |
| `role="row"`      | Set on the element; for usage inside the SwirlResourceList component.                           |
| `role="gridcell"` | Set on the inner container element; for usage inside the SwirlResourceList component.           |
| `role="button"`   | Implicit when used as a button.                                                                 |
| `role="link"`     | Implicit when used as a link.                                                                   |
| `role="checkbox"` | Set on the interactive part if used as a checkbox.                                              |
| `aria-disabled`   | If not used as a link, the `aria-disabled` attribute is set to `true` for the disabled element. |

### Keyboard

| Key              | Action                                                                                  |
| ---------------- | --------------------------------------------------------------------------------------- |
| <kbd>ENTER</kbd> | Activates the item if used as a button or link.                                         |
| <kbd>SPACE</kbd> | Activates the item if used as a button or link. Toggles the item if used as a checkbox. |
