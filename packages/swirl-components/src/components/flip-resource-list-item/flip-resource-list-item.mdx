# FlipResourceListItem

The FlipResourceListItem component can be used to represent an object, like
media, a group or a chat within a collection. It should only be used inside the
[FlipResourceList component](?path=/docs/components-flipresourcelist--flip-resource-list).

## Theming

| Variable                 | Usage                                |
| ------------------------ | ------------------------------------ |
| `--s-background-default` | Background color of the item         |
| `--s-background-hovered` | Background color of the hovered item |
| `--s-background-pressed` | Background color of the active item  |
| `--s-text-default`       | Text color of the item's label       |
| `--s-text-subdued`       | Text color of the item' description  |
| `--s-text-disabled`      | Text color of the disabled item      |
| `--s-border-default`     | Border color of the item             |

## Accessibility

### ARIA

Depending on the purpose, the item can be used as a button, link or checkbox
control.

| ARIA            | Usage                                                                                           |
| --------------- | ----------------------------------------------------------------------------------------------- |
| `role`          | Set according to the item usage.                                                                |
| `aria-disabled` | If not used as a link, the `aria-disabled` attribute is set to `true` for the disabled element. |

### Keyboard

| Key              | Action                                                                                  |
| ---------------- | --------------------------------------------------------------------------------------- |
| <kbd>ENTER</kbd> | Activates the item if used as a button or link.                                         |
| <kbd>SPACE</kbd> | Activates the item if used as a button or link. Toggles the item if used as a checkbox. |
