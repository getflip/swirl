# SwirlActionListItem

The SwirlActionListItem component is used to render menu items inside the
[SwirlActionList component](?path=/docs/components-swirlactionlist--docs).

## Theming

| Variable                      | Usage                                                   |
| ----------------------------- | ------------------------------------------------------- |
| `--s-surface-overlay-default` | Background color of the item                            |
| `--s-surface-overlay-hovered` | Background color of the hovered item                    |
| `--s-surface-overlay-pressed` | Background color of the active item                     |
| `--s-text-default`            | Text color of the item's label                          |
| `--s-text-subdued`            | Text color of the item's description and badge          |
| `--s-icon-default`            | Icon color of the item                                  |
| `--s-icon-highlight`          | Icon color of the item's suffix                         |
| `--s-text-critical`           | Text color of the item's label with intent "critical"   |
| `--s-icon-critical`           | Icon color of the item with intent "critical"           |
| `--s-text-disabled`           | Text color of the disabled item's label and description |
| `--s-icon-disabled`           | Icon color of the disabled item                         |

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Menu Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menu/).

| ARIA              | Usage                                                                                                            |
| ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| `role="menuitem"` | The action list item has `role` set to `menuitem`. It should only be used inside a component with `role="menu"`. |

### Keyboard

| Key              | Action              |
| ---------------- | ------------------- |
| <kbd>ENTER</kbd> | Activates the item. |
| <kbd>SPACE</kbd> | Activates the item. |
