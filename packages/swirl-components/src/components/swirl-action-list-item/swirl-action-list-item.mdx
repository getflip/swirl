[SwirlActionList component](?path=/docs/components-swirlactionlist--docs).

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Menu Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menu/).

| ARIA              | Usage                                                                                                            |
| ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| `role="menuitem"` | The action list item has `role` set to `menuitem`. It should only be used inside a component with `role="menu"`. |

### Keyboard

| Key              | Action              |
| ---------------- | ------------------- |
| <kbd>ENTER</kbd> | Activates the item. |
| <kbd>SPACE</kbd> | Activates the item. |
