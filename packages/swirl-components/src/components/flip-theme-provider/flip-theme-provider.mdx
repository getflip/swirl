# FlipThemeProvider

The FlipThemeProvider implements automatic theme switching with respect to the
user's OS theme setting, and allows to override the OS setting with a preferred
user theme. Via the configuration object of the provider, deviating tenant
colors can be set.

- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/flip-theme-provider)**
