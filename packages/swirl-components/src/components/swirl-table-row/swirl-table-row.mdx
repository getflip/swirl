# SwirlTableRow

The SwirlTableColumn component is used to display a row inside a
[SwirlTable component](?path=/docs/components-swirltable--docs).

## Related components

- **[swirl-table](?path=/docs/components-swirltable--docs)** (parent) — Table
  container; use `swirl-table-row` in the `rows` slot of `swirl-table` or inside
  `swirl-table-row-group`.
- **[swirl-table-cell](?path=/docs/components-swirltablecell--docs)** (child) —
  Table cells; use as direct children of `swirl-table-row`.

## Accessibility

### ARIA

The component follows the
[WAI-ARIA Table Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/table/).

| ARIA            | Usage                                                 |
| --------------- | ----------------------------------------------------- |
| `role="row"`    |                                                       |
| `aria-rowindex` | Set to the overall index of the row inside the table. |

### Keyboard

No component-specific keyboard behavior.
