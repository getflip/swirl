# SwirlToggleGroup

The SwirlToggleGroup component is used to group and manage the pressed state of
the [SwirlToggleButton](?path=/docs/components-swirltogglebutton--docs)
component. Only one option can be selected at a time.

- **[Source Code](https://github.com/flip-corp/swirl/tree/main/packages/swirl-components/src/components/swirl-toggle-group)**
